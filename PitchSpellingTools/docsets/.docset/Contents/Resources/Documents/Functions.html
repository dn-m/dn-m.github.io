Comparable
  • <(_:_:)

    Note

    In the case that both values are in the same octave, true is returned if the spelling of the SpelledPitch value on the left is less than that of the SpelledPitch on the right. This manages extreme scenarios such as (c#, dbb), which should have a named interval of a double diminished second, not a double augmented seventh.

    public func < (lhs: PitchSpelling, rhs: PitchSpelling) -> Bool

    Return Value

    true if the PitchSpelling value on the left has a lesser letterName value. In the case that the letterName values are equivalent, true if the PitchSpelling value on the left has a lesser quarterStep value. In the case that the quarterStep values are equivalent, true if the PitchSpelling value on the left has a lesser eighthStep value. Otherwise, false.
