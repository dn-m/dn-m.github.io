ArrayTools

Extensions of Swift Array struct

Usage

Destructure a list (Array) into Head and Tail, if possible

let emptyArray = []
let (head, tail) = emptyArray.destructured // nil

let array = [1,2,3]
let (head, tail) = array.destructured // (1, [2,3])

Construct a list (Array) from a Head and Tail

let head = 5
let tail = [10,15]
let list = head + tail  // [5,10,15]

Integration

Carthage

The best way to integrate ArrayTools into your iOS or OSX project is with Carthage.

Follow these instructions to install Carthage.

  1. Add ArrayTools to your Cartfile: github "dn-m/ArrayTools"
  2. Update Carthage from the command line: carthage update
