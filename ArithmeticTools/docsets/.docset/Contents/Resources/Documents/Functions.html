  • invert(powerOfTwo:within:)

    Takes an integer that is a power-of-two (e.g., 1 << 3). This retrieves the right-hand-side value, then inverts this value in relation to the given max value. The returned value can then be re-shifted as necessary.

    invert(powerOfTwo: 1 << 3, within: 4) // 1

    Declaration

    Swift

  • closer(to:a:b:)

    Get the closer of two values to a target value.

    closer(to: 11, a: 12, b: 3) -> 12

    Note

    If the distances between both values and the target value are equivalent, the first value (a) is returned.

    closer(to: 6, a: 4, b: 8) -> 4
