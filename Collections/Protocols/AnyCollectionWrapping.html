AnyCollectionWrapping

public protocol AnyCollectionWrapping: Collection

AnyCollectionWrapping is a type-erasing protocol that allows a Collection-conforming structure to wrap any underlying Collection implementation.

As a result, all of the Collection boilerplate is done for free.
